## boomJS

@(boomJS)[v1.0]
### 一、简介
- 基于jquery
- 使用ES6语法

### 二、demo
#### 1、使用
    $([selector]).imgBoom(option);
#### 2、参数说明
- columns：分割列数，数字
- row：分割行数，数字
- isBlur： 是否使用模糊效果（分割太多会卡顿）
- isBoom： 是否添加点击炸裂效果（会删除元素）
- duration：动画时长
